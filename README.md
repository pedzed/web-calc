# Web Calc
A simple web calculator based on Tres Framework.

## Features
- Calculations
    - Addition
    - Subtraction
    - Multiplication
    - Multiplication table
    - Division
    - X to the power of two
    - X to the power of Y
    - Square root
- Old results will be saved in POST
- Responsive design
